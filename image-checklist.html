Tips for Creating Images

Profile Image
  • Recommended size: 500x500 pixels (square)
  • Professional headshot with neutral background
  • Well-lit with good composition
Project Thumbnails
  • Recommended size: 800x600 pixels (16:9 aspect ratio)
  • Consistent style across all thumbnails
  • Visually represents the project's focus
  • High contrast for text overlay visibility
Favicon
  • Recommended size: 16x16, 32x32, 48x48 pixels
  • Simple design that works at small sizes
  • Save as .ico file format
